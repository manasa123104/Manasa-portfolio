## Enable GitHub Pages (2 minutes)

1. **Go to your repository:**
   - https://github.com/manasa123104/Manasa-portfolio

2. **Click on "Settings"** (top menu bar)

3. **Scroll down to "Pages"** in the left sidebar

4. **Under "Source":**
   - Select **Branch: main**
   - Select **Folder: / (root)**
   - Click **"Save"**

5. **Wait 2-5 minutes** for GitHub to build your site

6. **Your portfolio will be live at:**
   - **URL:** https://manasa123104.github.io/Manasa-portfolio/
